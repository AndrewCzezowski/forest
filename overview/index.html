            Definition. Elementary Vector Space [0003]

            • April 9, 2026

            A vector space is a set where elements can be added and scaled in a way that preserves structure. A vector is an element of a set called a vector space. We denote this set with \(\mathbb {V}\).

            Not every set is a vector space so we need properties which define a set as a vector space.

            1. Closure Under Addition

              The addition of two elements of a vector space will also belong to the vector space.

              \[\forall v,w \in \mathbb {V}, \exists \; v+w=s\in \mathbb {V}\]

              Vector addition is commutative and associative.

            2. Scalar Multiplication

              The operation of multiplying an element of the vector space by a scalar will yield another vector in the space.

              \[\forall \ v\in \mathbb {V} \land a\in \mathbb {F}, \exists \; av\in \mathbb {V}\]

              Scalar multiplication is distributive and associative in both the scalars and the vectors.

            3. The Scalar Field

              The scalars that multiply the vectors are called the scalar field \(\mathbb {F}\) over which the vector space is defined.

              Real vector spaces are characterized by only having members with scalar multiplication my the set of real numbers. Similar for complex vector spaces.

            4. Linearity

              \[\forall \ v, w\in \mathbb {V} \land a\in \mathbb {F}\rightarrow a(v+w)=av+aw\]

            5. Identity Under Vector Addition and Scalar Multiplication

              \(0+v=v\) and \(1v=v\)

            6. The Null Vector

              Every vector space must contain the null vector. Additionally, every vector has its opposite.

              \[\forall \;v\in \mathbb {V}, \exists \;w\in \mathbb {V}\;|\;w=-v\; \land v-v=0\]

            Dimensionality

            A consequence of closure under addition and scalar multiplication is that any vector in the space can be defined as a linear combination of other some other vectors, this leads us to the concept of Dimensionality. The minimum number of vectors that could possibly describe any vector in that space. In other words the number of basis vectors. Dimensionality can be infinite.

            Example. How to Make a Map [0004]

            • April 9, 2026

            Suppose we are working in a 4-dimensional real vector space, we can choose our basis vectors and label them as,

            \[e_\mu \in \mathbb {V},\ \mu =0,1,2,3\]

            Any vectors belonging to \(\mathbb {V}\) can now be written as,

            \[\vec {A}=A^\mu e_\mu \]

            Suppose \(\mathbb {W}\) is a vector space with potentially a different dimensionality. We can choose to define a linear map which takes \(\mathbb {V}\) to \(\mathbb {W}\),

            \[\Lambda :\mathbb {V}\rightarrow \mathbb {W}\]

            Here we use the notation,

            \[\Braket {\Lambda ,v}\rightarrow w\]

            Suppose \(\mathbb {W}\) has basis vectors \(f_\alpha \). To understand how the map applies to \(\mathbb {V}\) we need to know how each of the basis vectors \(e_\mu \) transforms in the map,

            \[\braket {\Lambda , e_\mu }=C_\mu {}^\alpha f_\alpha \]

            A linear map has the property,

            \[\braket {\Lambda , av_1+bv_2}=a\braket {\Lambda ,v_1}+b\braket {\Lambda ,v_2}\]

            The fact that the maps are linear means that given that we know how the basis vectors transform, we know how any vector in \(\mathbb {V}\) space maps to \(\mathbb {W}\),

            \[\braket {\Lambda , A^\mu e_\mu }=A^\mu \braket {\Lambda ,e_\mu }=A^\mu C_\mu {}^\alpha f_\alpha \]

            Definition. Dual Spaces [0005]

            • April 9, 2026

            Consider that we have defined a real vector space of dimension n (here we work in 4 dimensions) and we want a map, but we do not wish to create a second vector space.

            Since \(\mathbb {R}\) is included in our definition of a real vector space, we can consider mapping from \(\mathbb {V}\) to \(\mathbb {R}\), without creating anything new.

            We consider the set of all possible maps that take \(\mathbb {V}\rightarrow \mathbb {R}\). We call this set of the dual space and denote it by \(\mathbb {V}^*\).

            The dual space satisfies all of the properties of a vector space, so it is itself also a vector space. Since we did not add anything new to our original vector space, this means that by defining a vector space we automatically get an associated dual space.

            It is convenient (and arbitrary) to make the choice that the respective basis vectors map to one another as,

            \[\braket {e^\mu ,e_\nu }=\delta ^\mu _\nu \]

            So that,

            \[\braket {B_\mu e^\mu , A^\nu e_\nu }=B_\mu A^\mu \]

            Definition. Basis Transformations on Vector Spaces [0006]

            • April 9, 2026

            We start by considering an elementary vector space, with basis vectors \(e_\mu \). We can equivalently describe this vector space with some other basis vectors \(f_\nu \), so that a vector \(w\in \mathbb {V}\) can be written as,

            \[w=A^\mu e_\mu =B^\nu f_\nu \]

            We want to know how to convert between the old and new basis. We define the linear transformation:

            \[f_\nu =\Lambda _\nu {}^\mu e_\mu \]

            This is the same as the notion of change of basis in standard linear algebra. Matrices can be defined as the linear transformation inside vector spaces. \(\Lambda _\nu {}^\mu \) here is a matrix.

            \[\begin {bmatrix}f_0 \\ f_1\\ f_2\\ f_3\end {bmatrix}=\begin {bmatrix} \Lambda _{00} & \Lambda _{01} & \Lambda _{02}&\Lambda _{03}\\\Lambda _{10}&\Lambda _{11}&\Lambda _{12}&\Lambda _{13}\\\Lambda _{20}&\Lambda _{21}&\Lambda _{22}&\Lambda _{23}\\\Lambda _{30}&\Lambda _{31}&\Lambda _{32}&\Lambda _{33} \end {bmatrix}\begin {bmatrix}e_0 \\ e_1\\ e_2\\ e_3\end {bmatrix}\]

            Definition. Covariant and Contravariant Vectors [0007]

            • April 9, 2026

            Consider how a linear transformation between basis vectors affects an arbitrary vector,

            \[w=B^\nu f_\nu =B^\nu \Lambda _\nu {}^\mu e_\mu \implies A^\mu =B^\nu \Lambda _\nu {}^\mu \]

            We can also consider the inverse transformation,

            \[e_\mu =(\Lambda ^{-1})_\mu {}^\nu f_\nu \]

            leading to

            \[B^\mu =(\Lambda ^{-1})_\nu {}^\mu A^\nu \]

            Since the basis vectors \(f_\nu \) transform covariantly, we call them covariant vectors. On the other hand, since the coefficients \(B^\nu \) transorm with the inverse matrix, we say they transform contravariantly.

            We can extent this same idea to the dual space. We want to transform from the basis \(e^\mu \) to \(f^\nu \). From the above we know that \(e^\mu \) will transform contravariantly, so we can write,

            \[f^\nu =(\Lambda ^{-1})_\mu {}^\nu e^\mu \]

            and for an arbitrary dual vector,

            \[w=C_\nu e^\nu =D_\nu f^\nu \implies D_\nu =\Lambda _\nu {}^\mu C_\mu \]

            From this we note that \(f^\nu \) transforms contravariantly, so they are contravariant vectors. The coefficients \(D_\nu \) transform covariantly.

            Definition. Cartesian Product [0001]

            • April 8, 2026

            The cartesian product is a binary operation defined between two sets, that produces a set of all ordered pairs from the arguments.

            \[\set {a,b,c,...}\times \set {\alpha ,\beta ,\gamma ,...}=\Set {(a,\alpha ),(a,\beta ),(a,\gamma ),...,(b,\alpha ),(b,\beta ),(b,\gamma ),...}\]

            or

            \[A\times B=\set {All\; ordered\; pairs\; of A\;and\;B}\]

            Note: Since the cartesian product is a set, we are not limited to only creating binary operations. We can go to arbitrarily higher orders.

            \[A\times B\times C\times D\]

            Definition. Tensor Products [0008]

            • April 9, 2026

            Suppose we want to consider a map, which takes the cartesian product of a vector space with itself to the real numbers. In other words a binary operation on the vector space to the real numbers.

            \[\_\_:V\times V\rightarrow \mathbb {R}\]

            We do not want to create anything new for our map, so the obvious choice is to draw the map from the dual space.

            \[\set {v,w,...}\in V\; and\; \set {\alpha ,\beta ,...}\in V^*\]

            We define the tensor product, \(\beta \otimes \alpha \), which linearly maps an ordered pair of numbers to \(\mathbb {R}\)

            \[\left [\beta \otimes \alpha \right ] (v,w)=\braket {\beta ,v}\braket {\alpha ,w}\]

            and using its linearity,

            \[[a\beta \otimes \alpha +b\gamma \otimes \delta ] (v,w)=a[\beta \otimes \alpha ] (v,w)+b[\gamma \otimes \delta ] (v,w)\]

            If we consider the set of all possible tensor products of covectors: we have a linear space, with the notion of addition, and scalar multiplication; which satisfies all of the properties of a vector space. So, all tensor product spaces are vector spaces; and since the elements of a vector space are vectors, all tensors are vectors. Such a space is also called a Tensor Product Space.

            An arbitrary tensor can be written as \(\beta \otimes \gamma \equiv T_{\mu \nu }e^\mu \otimes e^\nu \)

            This idea can be easily generalized to consider higher order and mixed tensors. For instance,

            \[V^*\otimes V^*\otimes V^*:V\times V\times V \rightarrow \mathbb {R}\]

            Definition. Rank of a Tensor Product Space [0009]

            • April 9, 2026

            What we call tensors are objects where vectors are first and dual vectors are second. This does not mean that objects that do not follow this ordering are illogical, but we do not assign them a rank as a tensor.

            The idea of a rank:

            • A vector: \(A^\mu e_\mu \rightarrow \) rank (1,0)
            • A covector: \(A_\mu e^\mu \rightarrow \)rank (0,1)
            • \(T^{\mu \nu \delta \lambda }e_\mu \otimes e_\nu \otimes e_\delta \otimes e_\lambda \rightarrow \) rank (4,0)
            • \(T_{\mu \nu \delta \lambda }\;e^\mu \otimes e^\nu \otimes e^\delta \otimes e^\lambda \rightarrow \)rank (0,4)
            • \(T^{\mu \nu }{}_{\delta \lambda }\;e_\mu \otimes e_\nu \otimes e^\delta \otimes e^\lambda \rightarrow \)rank (2,2)
            • ...

            We use the notation,

            \[V\otimes V^*\otimes V^*\equiv \mathcal {T}^1{}_2=\set {All\; possible\; tensors\; of\; rank\; (1,2)}\]

              Definition. Euler Lagrange Equations [000B]

              • April 12, 2026

              The Euler-Lagrange equations, \[\frac {\partial \mathcal {L}}{\partial q_i}=\frac {d}{dt}\frac {\partial \mathcal {L}}{\partial \dot {q}_i}\] are a condition that is satisfied for a path that gives a stationary value for an equation of the form, \[S=\int _{t_1}^{t_2}\mathcal {L}(q_1,\dot {q}_1,\dots ,q_n,\dot {q}_n,t)dt\] Where \(q_i\) is the \(i\)th generalized coordinate parametrized by an independent variable \(t\), and \(\dot {q}_i\) is the first derivative of \(q\) with respect to \(t\).

              Proof

              For a full proof see Chapter 6 of Classical Mechanics by Taylor. In summary, one starts by considering variations from the correct path (the correct path gives a stationary point for S). After computing the variation \(\delta S\), integrating by parts, applying boundary conditions, requiring \(\delta S=0\), and that coordinates vary independently; we find the condition of the Euler-Lagrange equations.

              Definition. Hamilton's Principle [0011]

              • April 16, 2026

              The motion of a system from time \(t_1\) to \(t_2\) is such that the path integral (called the action), \[S=\int _{t_1}^{t_2}Ldt\] is stationary for the actual path of the motion. Where the lagrangian \(L=T-U\) is the difference between kinetic and potential energy.

              This principle applies for systems that are monogenic (all non-constraint forces are derivable from a scalar potential that can only depend on coordinates, velocities, and time).

              Concept. Lagrange Equations [0012]

              • April 16, 2026

              We know that the conditions for an integral of the form of Hamilton's Principle to be stationary are the Euler-Lagrange equations, \[\frac {\partial L}{\partial q_i}-\frac {d}{dt}\frac {\partial L}{\partial \dot {q}_i}=0\qquad i=1, \dots , n\] Where \(q_i\) is the \(i\)th generalized coordinate and all generalized coordinates vary independently.

              Because of how they behave we call, \[\frac {\partial L}{\partial q_i}=(i\text {th component of generalized force})\] and \[\frac {\partial L}{\partial \dot {q}_i}=(i\text {th component of generalized momentum})\]

              This formulation works without modification for systems that are holonomic (constraints only of the form \(f_\alpha (q_1,\dots ,q_n;t)=0\)). This includes systems that are unconstrained. The condition that the system be holonomic is what allows us to write the \(n\) generalized coordinates of the system as independent.

              If the system is constrained holonomically, then we can write the dependent coordinates in terms of others using the constraint functions. This leave us with only independent generalized coordinates.

              Concept. Method of Lagrange Multipliers [0013]

              • April 17, 2026

              For a holonomically (also possible for semi-holonomic) constrained system, rather than choose independent generalized coordinates like in the standard Lagrange Equations; we can choose to work with a larger number of coordinates and off-load the constraints to Lagrange Multipliers.

              For a system with \(n\) variables and \(m\) constraint equations \(f_\alpha (q_1,\dots ,q_n;t)=0\), the resulting equations are, \[\frac {\partial L}{\partial q_i}-\frac {d}{dt}\frac {\partial L}{\partial \dot {q}_i}+\sum _{\alpha =1}^{m}\lambda _\alpha \frac {\partial f_\alpha }{\partial q_i}=0\qquad i=1,\dots ,m\] Where \(\lambda _\alpha \) are the lagrange multipliers.

              Additionally, we can identify, \[\sum _{\alpha =1}^{m}\lambda _\alpha \frac {\partial f_\alpha }{\partial q_i}=Q_i\] Where \(Q_i\) is the magnitude of the \(i\)th generalized force required to produce the constraint.

              Proof

              Start by considering the variation of the action the standard way; which can be written as, \[\delta S=\int _{t_1}^{t_2}dt\sum _{i=1}^{n}\left (\frac {\partial L}{\partial q_i}-\frac {d}{dt}\frac {\partial L}{\partial \dot {q}_i}\right )\delta q_i=0\] The difference in this case is that we cannot vary \(q_i\)'s independently, which is what allowed us to come to the Euler-Lagrange equations in the first place.

              Consider the holonomic constraint functions, \[f_\alpha (q_1,\dots ,q_n;t)=0 \qquad \alpha =1,\dots ,m\] Since each equation is equal to zero we can multiply it by an arbitrary function \(\lambda _{\alpha }(t)\) \[\lambda _\alpha (t)f_\alpha (q_1,\dots ,q_n;t)=0\] and consider the variation, \[\delta f_\alpha =\sum _{\alpha =1}^{m}\lambda _{\alpha }\frac {\partial f_\alpha }{\partial q_i}\delta q_i=0\] Important to note is that this variation is of the coordinates and not time just like in the action integral.

              Since this equation is zero we can add it to the action variation without changing the value, \[\delta S=\int _{t_1}^{t_2}dt\sum _{i=1}^{n}\left (\frac {\partial L}{\partial q_i}-\frac {d}{dt}\frac {\partial L}{\partial \dot {q}_i}+\sum _{\alpha =1}^{m}\lambda _{\alpha }\frac {\partial f_\alpha }{\partial q_i}\delta q_i\right )\delta q_i=0\]

              Since at this point \(\lambda _\alpha \) is arbitrary we can choose them so that \(m\) of the equations are satisfied for arbitrary \(\delta q_i\), and then choose the remaining \(n-m\) equations independently. Which gives us, \[\frac {\partial L}{\partial q_i}-\frac {d}{dt}\frac {\partial L}{\partial \dot {q}_i}+\sum _{\alpha =1}^{m}\lambda _\alpha \frac {\partial f_\alpha }{\partial q_i}=0\qquad i=1,\dots ,m\]

              It is possible to extend the ideas presented above to include semi-holonomic constraints \(f_\alpha (q_1,\dots ,q_n;\dot {q}_1,\dots ,\dot {q}_n;t)\). However, it is in general not possible to express non-holonomic contraints via a variational principle.

              Definition. Ignorable Coordinates [0014]

              • April 17, 2026

              In the Lagragian formulation a generalized coordinate is called ignorable or cyclic if the lagragian is independent of that coordinate.

              Consider the case that the lagrangian is independent of the coordinate \(q_k\), \[\frac {\partial L}{\partial q_k}=0\] and by the lagrange equations this leads to \[\frac {d}{dt}\frac {\partial L}{\partial \dot {q}_k}=0\] Implying that the generalized momentum corresponding to coordinate \(q_k\) is conserved.

                Definition. Field [000G]

                • April 15, 2026

                A field is any set where the following properties hold.

                1. Addition and multiplication of the elements is well defined.

                2. Addition and multiplication is commutative, associative, and distributive.\[a(b+c)=ab+ac\]

                3. The set must include an additive and a multiplicative identity.\[a+0=a\qquad \qquad 1a=a\]

                4. Every element must have an additive and a multiplicative inverse.\[a-a=0\qquad \qquad a\frac {1}{a}=1\]

                As an example, the rational numbers qualify as a field; the integers do not.

                Definition. Subspaces [000C]

                • April 12, 2026

                A subset \(U\) of a vector space \(\mathbb {V}\), is a subspace if \(U\) is also a vector space with the same additive identity, addition, and scalar multiplication as on \(V\).

                In practice to check if a subset \(U\) of \(\mathbb {V}\) is a subspace one must check that \(U\) satisfies:

                1. Additive Identity \[0\in U\]

                2. Closure Under Addition \[u,w \in U\rightarrow u+w\in U\]

                3. Closure Under Scalar Multiplication \[a\in \mathbb {F} \land u\in U\rightarrow au\in U\]

                Definition. Sum of Subspaces [000D]

                • April 12, 2026

                The sum of subspaces \(V_1,\dots ,V_n\) belonging to a vector space \(\mathbb {V}\) is the set of all possible sums of elements from \(V_1,\dots ,V_n\). \[V_1+\dots +V_n=\{v_1+\dots +v_n:v_1\in V_1,\dots v_n\in V_n\}\]

                Lemma. Smallest Containing Subspace [000E]

                • April 12, 2026

                Consider subspaces \(V_1,\dots ,V_n\) belonging to a vector space V. The smallest subspace of V that contains \(V_1,\dots ,V_n\) is the sum of subspaces \(V_1+\dots +V_n\).

                For proof it is easy to see that the sum contains each individual subspace and is itself a subspace of V. Now consider that due to closure under addition any other subspace containing \(V_1,\dots ,V_n\) also contains \(V_1+\dots +V_n\). So any other space containing these will be larger or the same.

                Definition. Direct Sum [000F]

                • April 12, 2026

                The direct sum is a special case of the sum of subspaces. If each element from the sum \(V_1+\dots +V_n\) can only be written as a sum of \(v_1+\dots +v_n\), where each \(v_k\in V_k\). We call it a direct sum and denote it by, \[V_1\oplus \dots \oplus V_n\]

                Condition for a Direct Sum

                The sum of subspaces, \(V_1+\dots +V_n\), is a direct sum \(\iff \) the only way to write 0 as a sum \(v_1+\dots +v_n\) is by taking each element equal to zero.

                Another way to check

                The sum of subspaces, \(V_1+V_2\), is a direct sum \(\iff \) \(V_1\cap V_2=\{0\}\).

                Definition. Span and Spans [0016]

                • April 17, 2026

                The span of a list of vectors in \(V\) is denoted and defined by, \[\text {span}(v_1,\dots ,v_m)=\{a_1 v_1+\dots +a_m v_m;a_m \in \mathbb {F} a\}\] We define the span of the empty list as \(\{0\}\).

                It is easy to see that the span of a list of vectors in \(V\) is the smallest subspace cantaining all of the listed vectors.

                If the span of a list of vectors equals \(V\), we say that the list spans \(V\).

                Definition. Finite Dimensional Vector Space [0017]

                • April 17, 2026

                A vector space \(V\) is finite dimensional if there exists a list of vectors that spans \(V\). Note that by definition lists are finite.

                Definition. Infinite Dimensional Vector Space [0019]

                • April 17, 2026

                A vector space \(V\) is infinite dimensional it is not a finite dimensional vector space. In other words there does not exist a list of vectors that spans \(V\).

                Definition. Linearly Independent Vectors [001A]

                • April 17, 2026

                A list of vectors \(v_1,\dots ,v_m\in V\) is called linearly independent if the only way to write \(0\) as a linear combination of those vectors is by taking all the coefficients to be zero. We also define the empty list to be linearly independent.

                An equivalent statement for a list of linearly independent vectors is that any vector in the span of \(v_1,\dots ,v_m\in V\) has a unique representation as a linear combination of those vectors.

                Definition. Linearly Dependent Vectors [001B]

                • April 17, 2026

                A list of vectors \(v_1,\dots ,v_m\in V\) is called linearly dependent if we can write \(0\) as a linear combination of those vectors without taking all the coefficients to be zero.

                Equivalently, this means that we can describe some of the vectors in the list as linear combinations of the others.

                A list of vectors is linearly dependent if it is not linearly independent.

                Lemma. Linear Dependence Lemma [001C]

                • April 17, 2026

                Consider a linearly dependent list of vectors \(v_1,\dots ,v_m\in V\). There exists \(k\in \{1,\dots ,m\}\) such that, \[v_k\in \text {span}(v_1,\dots ,v_{k-1})\] and if the \(k\)th term is removed from the list, the span of the remaining list is unchanged.

                These statements are both easy to consider, since if the vectors are linearly dependent; there will be a \(v_k\) that can be writen as a linear combination of the vectors before it. Clearly since \(v_k\) can be described in terms of those other vectors the span will be unchanged it we remove it from the list.

                Definition. Finite Dimensional Basis [001D]

                • April 17, 2026

                A basis of \(V\) is a list of vectors in \(V\) that is linearly independent and spans \(V\). A list is a basis if and only if every vector in the space can be written as a unique linear combination of the vectors in that list.

                It is easy to see that any spanning list in a vector space can be reduced to a basis by removing dependent vectors until we are left with a list of independent ones.

                This means that any finite-dimensional vector space will have a basis.

                Similarly, we can create a basis from any list of independent vectors by adding independent vectors until we can span the whole space.

                If we can find a spanning list with length equal to the dimension of the vector space, then we have found the basis of that space.

                Definition. Dimension of a Finite Vector Space [0018]

                • April 17, 2026

                The dimension of a finite dimensional vector space is the length of any basis of the vector space. Important to this definition is the fact that every basis of a given space has the same length.

                  Definition. Function [000H]

                  • April 15, 2026

                  Given two sets \(A\) and \(B\), a function from \(A\) to \(B\) is a mapping that takes each element \(x\in A\) to a single element of \(B\). \[f:A\rightarrow B\]

                  \(f(x)\) represents the element of \(B\) associated with \(x\) by \(f\).\[B=\{y\in B: y=f(x) \text { for some } x\in A\}\]

                  The domain of \(f\) is the set \(A\). The range of \(f\) is some subset of \(B\), not necessarily equal to \(B\).

                  Definition. Simple Definition for The Real Numbers [000J]

                  • April 15, 2026

                  To provide an initial definition for \(\mathbb {R}\), we assume that \(\mathbb {R}\) is an ordered field which contains \(\mathbb {Q}\) as a subfield. We also require that it contain no gaps between rational numbers. We accomplish this second requirement via the axiom of completeness.

                  Definition. Supremum and Infimum [000K]

                  • April 15, 2026

                  A supremum is the least upper bound of a set. Likewise, the infimum is the greatest lower bound of a set.

                  A real number \(s\) is the least upper bound of a set \(A\subseteq \mathbb {R}\) if it meets the requirements:

                  1. \(s\) is an upper bound for \(A\).
                  2. If \(b\) is any upper bound for \(A\), then \(s\leq b\).

                  There is a matching definition for the least lower bound.

                  The first time I read about these I was confused due to my misunderstanding of an upper/lower bound.

                  A set \(A\subseteq \mathbb {R}\) is bounded above if there exists a number \(b\in \mathbb {R}\) such that \(a\leq b\) for all \(a\in A\). The number b is called an upper bound for \(A\).

                  The key point is that an upper bound does not need to be in a set; it is simply any number that is greater than or equal to any elements of the set. This notion becomes clear when considering open intervals where a maximum in the set does not exist, but the supremum does (it's the open upper bound). For a closed interval the maximum and supremum are the same.

                  Theorem. Nested Interval Property [000M]

                  • April 15, 2026

                  For each \(n\in \mathbb {N}\), assume we are given a closed interval \(I_n=[a_n,b_n]=\{x\in \mathbb {R}:a_n\leq x\leq b_n\}\). Assume that each interval \(I_n\) contains \(I_{n+1}\). Then, the resulting nested sequence of closed intervals \[I_1\supseteq I_2 \supseteq I_3 \supseteq \dots \] has a non empty intersection; \[\bigcap _{n=1}^{\infty } I_n\neq \emptyset \]

                  Proof

                  The idea in this proof is to consider a set, \(A=\{a_n:n\in \mathbb {N}\}\), where \(a_n\) are the lower bound of some closed intervals \([a_n, b_n]\). Since, \(b_n\) are upperbounds we know that the set it bounded from above and we can use the axiom of completeness to write \[x=\text {sup}A\] Since \(x\) is a least upper bound we can write \(a_n\leq x \leq b_n\), meaning that \(x\in I_n\) for any \(n\). This leads to the conclusion that the intersection of nested intervals is not empty.

                  Theorem. Archimedean Property [000N]

                  • April 15, 2026

                  1. Given any number \(x\in \mathbb {R}\), there exists an \(n\in \mathbb {N}\) satisfying \(n>x\).
                  2. Given any real number \(y>0\), there exists an \(n\in \mathbb {N}\) satisfying \(1/n<y\).

                  Proof

                  The theorem boils down to the statement that \(\mathbb {N}\) is not bounded above. The proof is done by contradiction. Assume that \(\mathbb {N}\) is bounded above; then the AoC says there will be a least upper bound. \[\alpha =\text {sup}\mathbb {N}\] Now consider \(\alpha -1\) which is not an upper bound, meaning that we can write, \[\alpha -1<n\qquad \rightarrow \qquad \alpha <n+1\] However, since \(n+1\in \mathbb {N}\), we have a contradiction for \(\alpha \) being an upper bound. Therefore, we come to the conclusion that \(\mathbb {N}\) is unbound from above.

                  Theorem. Density of Rational Numbers in The Reals [000O]

                  • April 15, 2026

                  For any two real numbers \(a\) and \(b\) with \(a<b\), there exists a rational number satisfying \[a<\frac {m}{n}<b\] In other words, we say that \(\mathbb {Q}\) is dense in \(\mathbb {R}\).

                  Proof

                  We are trying to find an \(m\in \mathbb {Z}\) and \(n\in \mathbb {N}\) so that, \[a<\frac {m}{n}<b \qquad \text {or} \qquad na<m<nb\] Using the Archimedean Property, we can pick an \(n\) large enough that \[a<b-\frac {1}{n}\] We now pick an integer \(m\) so that \[m-1\leq na< m\] Together with the previous inequality we can write use this to write, \[na<m<nb\] Proving the theorem.

                  Definition. One-to-One and Onto [000R]

                  • April 15, 2026

                  A function \(f:A\rightarrow B\) is one-to-one if any \(a_1\neq a_2\) in \(A\) implies that \(f(a_1)\neq f(a_2)\) in \(B\). The function \(f\) is onto if, given any \(b\in B\), it is possible to find an element \(a\in A\) for which \(f(a)=b\).

                  Put simply every element in \(B\) needs a corresponding element in \(A\) which maps uniquely via the function to \(B\).

                  The terminology injective (one-to-one), surjective (onto), and bijective (one-to-one and onto) are often used. There is also the confusing terminology one-to-one correspondence, which has the same meaning as bijection, rather than one-to-one function.

                  If there exists a function \(f:A\rightarrow B\) that is 1-1 and onto, we say that set \(A\) has the same cardinality as set \(B\).

                  Definition. Countable Sets [000S]

                  • April 15, 2026

                  An infinite set \(A\) is called countable if it has the same cardinality as the natural numbers. An infinite set that is not countable is called an uncountable set.

                  An example of a countable infinite set is the rational numbers. Examples of uncountable sets are the real numbers or the irrational numbers.

                  Countability and Subsets

                  If \(A\subseteq B\) and \(B\) is countable, then \(A\) is either countable or finite.

                  Countability and Unions

                  1. If \(A_1, \dots , A_m\) are each countable sets, then the union \(A_1\cup \dots \cup A_m\) is countable.
                  2. If \(A_n\) is a countable set for each \(n\in \mathbb {N}\), then \(\bigcup _{n=1}^{\infty } A_n\) is countable.

                  Definition. Power Sets [000T]

                  • April 15, 2026

                  The power set \(P(A)\) of a set \(A\), is a set whose elements are all of the different possible subsets of A.

                  For a finite set \(A\) with n elements, \(P(A)\) has \(2^n\) elements. This is because we can make every subset of \(A\) by considering a binary choice for each element on whether or note to include it in the set.

                  Theorem. Cantor's Theorem [000U]

                  • April 15, 2026

                  Given any set \(A\), there does not exist a function 1-1 and onto that maps \(f:A\rightarrow P(A)\). Where \(P(A)\) is the power set of \(A\).
